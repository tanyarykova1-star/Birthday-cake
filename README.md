# 🎂 Python Birthday Greeting

A heartwarming scripted animation for my Dad's birthday. Watch the celebration unfold right in your console or window!

## ✨ What happens inside:
* **Step-by-step rendering**: Watch as the cake, candles, and balloons appear one by one.
* **Animated Text**: Greeting message appears with a typewriter effect.
* **Pure Python**: No heavy libraries, just clean logic.

## 🚀 How to run
1. Make sure you have **Python** installed.
2. Download the file.
3. Run it via terminal

## 🎁 Open Source Template
I made this for my Dad, but feel free to use it as a template for your own family and friends. Just change the text in the code!
